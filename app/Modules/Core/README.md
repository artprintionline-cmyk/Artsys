# Módulo Core

Contém autenticação, usuários e empresas.

Estrutura:
- `Controllers/` — controllers do módulo
- `Models/` — modelos específicos do módulo (opcionais)
- `Migrations/` — migrations locais do módulo (opcionais)

Observação: no esqueleto principal os modelos essenciais (`User`, `Empresa`) foram colocados em `app/Models` para serem usados globalmente.
